# S206L1
Codes developed on Software Quality classes

# Arquivos Principais
cypress/e2e/first_ui_test.cy.js: onde se encontra a suíte de testes.  
cypress/reports/html/index.html: html com o relatório da suíte de testes.  


# Execução
1. No diretório desejado, clone este repositório com o comando no terminal:  

    git clone https://github.com/LannaCeS/S206L1.git  

2. Abra a pasta do projeto e, em um terminal dedicado insira o comando:  

    ./node_modules/.bin/cypress run --spec 'cypress/e2e/**/'
